## ğŸš€ Installation

1.  Open **Visual Studio Code**.
2.  Go to the **Extensions** view (`Ctrl+Shift+X` or `Cmd+Shift+X`).
3.  Search for `Elegant Sapphire` (published by **DanielHdez22**).
4.  Click **Install**.
5.  Once installed, go to **File > Preferences > Theme > Color Theme** and select **Elegant Sapphire**.
